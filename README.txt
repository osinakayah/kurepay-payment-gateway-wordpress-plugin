== Description ==

Description
This is a Kurepay payment gateway for Wordpress.

Kurepay is on a mission to deliver fast and convenient payment experience for customers and merchants.
Kurepay provides an infastructure that lets customers pay for goodd and services with both their credit card or cryptocurrencies without the merchant having to worry about the volatility of liquidity of the cryptocurrency.


To signup for a Kurepay Merchant account visit their website by clicking <a href="https://payment.kurepay.com">here</a>

With this Kurepay Wordpress Payment Gateway plugin, you will be able to accept the following payment methods in your shop:

Mastercard
Visa
Bitcoin
Ethereum
